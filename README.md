# Api-falabella

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/milts123/Api-falabella.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Api-falabella
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Set up environment variables as needed.

## Usage Guide

To start the application:
```bash
npm start
```
